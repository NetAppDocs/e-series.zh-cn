= I/O 模块维护概述 - EF50 和 EF80
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
EF50 和 EF80 存储系统在更换或升级 I/O 模块方面提供了灵活性，以增强网络连接和性能。

* link:io-module-replace.html["更换 I/O 模块"]
+
当存储系统中的 I/O 模块出现故障时，可以将其替换为相同类型的 I/O 模块，以将存储系统恢复到其最佳操作状态。

* link:io-module-upgrade.html["升级 I/O 模块"]
+
您可以升级存储系统中的 I/O 模块以增加主机端口数量，从而增强网络连接并扩展系统处理数据流量的能力。
