= 主机接口卡
:icons: font
:imagesdir: ../media/


[role="lead"]
可以选择将主机接口卡（ HIC ）安装在控制器箱中。E5700 控制器在控制器卡本身上包括内置主机端口，并在可选 HIC 上包括主机端口。控制器中内置的主机端口称为基板主机端口。HIC 中内置的主机端口称为 HIC 端口。
