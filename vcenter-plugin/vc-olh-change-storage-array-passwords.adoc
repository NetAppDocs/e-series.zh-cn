= 更改存储阵列密码
:hardbreaks:
:allow-uri-read: 
:experimental: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以在适用于vCenter的存储插件中更新用于查看和访问存储阵列的密码。

.开始之前
您必须知道在System Manager中设置的存储阵列的当前密码。

.关于此任务
在此任务中、您可以输入存储阵列的当前密码、以便在插件中访问该密码。如果在System Manager中更改了阵列密码、则可能需要执行此操作。

.步骤
. 从*管理*页面中、选择一个或多个存储阵列。
. 选择菜单：不常见任务(提供存储阵列密码)。
. 输入每个存储阵列的密码、然后单击*保存*。
