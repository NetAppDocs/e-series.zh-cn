= 为什么我看不到所有存储阵列？
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
在导入设置操作期间、某些存储阵列可能在目标选择对话框中不可用。

存储阵列可能不会显示、原因如下：

* 固件版本低于8.50。
* 存储阵列已脱机。
* 系统无法与该阵列进行通信(例如、该阵列存在证书、密码或网络问题)。
