= 打开定位灯
:hardbreaks:
:allow-uri-read: 
:experimental: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
您可以找到驱动器、以物理方式识别构成选定池、卷组或SSD缓存的所有驱动器。选定池、卷组或SSD缓存中的每个驱动器上都会亮起一个LED指示灯。

.步骤
. 从管理页面中，选择存储阵列。
. 选择菜单：配置[配置池和卷组]。
. 选择要查找的池、卷组或SSD缓存、然后单击菜单：更多(打开定位灯)。
+
此时将显示一个对话框、指示包含选定池、卷组或SSD缓存的驱动器上的指示灯已打开。

. 成功找到驱动器后、单击*关闭*。
