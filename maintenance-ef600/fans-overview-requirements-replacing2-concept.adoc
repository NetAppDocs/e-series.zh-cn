= 更换 EF300 或 EF600 风扇的要求
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在更换 EF300 或 EF600 阵列中的故障风扇之前，请查看以下要求。

* 您的控制器架或驱动器架型号支持更换风扇。
* 您已安装 ESD 腕带，或者已采取其他防静电预防措施。



NOTE: 如果 Recovery Guru 指示无法卸下风扇，请联系技术支持。
