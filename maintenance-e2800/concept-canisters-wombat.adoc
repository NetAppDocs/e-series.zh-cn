= 箱
:icons: font
:imagesdir: ../media/


[role="lead"]
箱由三种不同类型组成：为 12 驱动器或 24 驱动器控制器架或驱动器架提供冗余电源和充分散热的电源风扇箱（电源）；为 60 驱动器控制器架或驱动器架提供电源冗余的电源箱； 和风扇箱，用于冷却 60 驱动器控制器架或驱动器架。
