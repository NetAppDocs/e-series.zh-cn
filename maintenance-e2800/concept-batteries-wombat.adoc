= 电池
:icons: font
:imagesdir: ../media/


[role="lead"]
每个控制器箱都包含一个电池，用于在交流电源发生故障时保留缓存的数据。如果 SANtricity System Manager 中的恢复 Guru 报告电池故障状态或需要更换电池状态，则必须更换受影响的电池。
