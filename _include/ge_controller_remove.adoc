验证缓存内存中的任何数据是否已写入驱动器，从受损控制器拔下所有电缆，然后从机箱中取出受损控制器。

.步骤
. 在受损的控制器上，确保 NV Caching Active（绿色）LED 熄灭。
+
当 NV Caching Active（绿色）LED 熄灭时，缓存存储器中的任何数据都已写入驱动器，可以安全地移除受损的控制器。

+

NOTE: 如果 NV Caching Active（绿色）LED 指示灯亮起，则缓存数据正在写入驱动器。您必须等待过程完成，并且 NV Caching Active LED 指示灯关闭。但是，如果 LED 指示灯持续亮起超过五分钟，请在继续此过程之前联系 https://mysupport.netapp.com/site/global/dashboard["NetApp 支持"]。

+
NV 缓存活动（绿色）LED 位于控制器上的 NV 图标旁边。

+
image::../media/drw_g_nvmem_led_ieops-1839.svg[NV 状态 LED 位置]

+

. 戴上 ESD 腕带或采取其他防静电预防措施。
. 标记连接到受损控制器的每根电缆。
. 从受损控制器的电源上拔下电源线。
+

NOTE: 电源 (PSU) 没有电源开关。

. 从受损控制器上拔下所有电缆。
. 卸下受损的控制器：
+
下图显示了卸下控制器时控制器手柄的操作（从控制器的左侧）：

+
image::../media/drw_g_and_t_handles_remove_ieops-1837.svg[控制器句柄操作以删除控制器]

+
[cols="1,4"]
|===


 a| 
image::../media/icon_round_1.png[标注编号 1]
 a| 
在控制器的两端，向外推动垂直锁定凸耳，将手柄释放到水平位置。



 a| 
image::../media/icon_round_2.png[标注编号 2]
 a| 
** 将手柄拉向您，将控制器从中间背板上卸下。
+
当您拉动时，手柄从控制器伸出，然后您感到一些阻力，继续拉动。

** 将控制器滑出机箱，同时支撑控制器底部，并将其放置在平坦、无静电的工作表面上。




 a| 
image::../media/icon_round_3.png[标注编号 3]
 a| 
如果需要，将手柄向上旋转（选项卡旁边），将它们移开。

|===
. 逆时针旋转指旋螺钉以松开控制器盖，然后打开盖子。
