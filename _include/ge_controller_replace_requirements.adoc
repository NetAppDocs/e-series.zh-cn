* 确认您的控制器出现故障。您可以通过两种方式确定：
+
** SANtricity System Manager 中的 Recovery Guru 指示您更换控制器。
** 控制器上的琥珀色注意 LED 亮起，表示控制器出现故障。




* 验证存储系统的以下条件是否正确，以确保在存储系统开机并执行主机 I/O 操作时可以更换受损的控制器：
+
** 机箱中的第二个控制器具有最佳状态。
** SANtricity System Manager 的 Recovery Guru 详细信息区域中的 *OK to remove* 字段显示 *Yes*，表明删除受损控制器是安全的。


* 确保您已具备以下条件：
+
** 与要更换的控制器具有相同部件号的替换控制器。
** ESD 腕带，或者您已采取其他防静电预防措施。
** 一个无静电的平面。
** 用于识别要从控制器断开的每根电缆的标签。
** 具有浏览器的管理站，可以访问控制器的 SANtricity System Manager。
+
您可以通过将浏览器指向控制器的域名或 IP 地址来打开 SANtricity System Manager 界面。

+
您也可以使用命令行界面 (CLI) 执行某些过程。使用 SANtricity System Manager 下载 CLI 包（zip 文件）：*设置* > *系统* > *附加组件*，然后选择*命令行界面*。然后，您可以从操作系统提示符发出 CLI 命令，例如 DOS C: 提示符。
