.步骤
. 如果不知道存储系统中主机 I/O 模块上的协议，请使用 SANtricity System Manager 进行验证。
+
.. 选择 *Settings* > *System*。
.. 在 *其他设置* 部分，选择 *更改主机 I/O 协议* 以打开该页面。
.. 请参阅 *Host I/O Protocol* 字段中显示的默认协议。
+

NOTE: 如果要查看主机 I/O 模块的可用协议列表，请选择 *Host I/O Protocol* 字段中的下拉菜单。协议列表基于当前安装的主机 I/O 模块。



. 下一步取决于默认协议是否是您想要的协议：
+
[cols="1,2"]
|===
| 条件 | 操作 


 a| 
默认协议是您想要的协议
 a| 
无需采取操作。您可以退出此过程。



 a| 
默认协议不是您想要的协议
 a| 
继续此过程以更改协议。

|===
. 请为此过程安排一个停机维护窗口。
