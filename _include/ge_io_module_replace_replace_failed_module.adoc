.步骤
. 如果您尚未接地，请正确接地。
. 从出现故障的 I/O 模块中拔下电缆。
+
请务必给电缆贴上标签，以便了解电缆的来源。

. 从控制器上卸下发生故障的 I/O 模块：
+
image::../media/drw_g_io_module_replace_ieops-1900.svg[移除 I/O 模块]

+
[cols="1,4"]
|===


 a| 
image::../media/icon_round_1.png[标注编号 1]
 a| 
逆时针旋转 I/O 模块指旋螺钉以松开。



 a| 
image::../media/icon_round_2.png[标注编号 2]
 a| 
使用左侧的端口标签选项卡和指旋螺钉将 I/O 模块从控制器中拉出。

|===
. 拆开更换 I/O 模块的包装，并将其放置在存储系统附近的平坦无静电表面上。
+
保存包装材料，以便在退回有故障的 I/O 模块时使用。

. 将更换 I/O 模块安装到目标插槽中：
+
.. 将 I/O 模块与插槽的边缘对齐。
.. 轻轻将 I/O 模块完全推入插槽，确保将模块正确插入连接器。
+
您可以使用左侧的选项卡和指旋螺钉推入 I/O 模块。

.. 顺时针旋转指旋螺钉以拧紧。


. 连接 I/O 模块电缆。
