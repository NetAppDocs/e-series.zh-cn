.步骤
. 如果您尚未接地，请正确接地。
. 找到 NV 电池。
. 取下 NV 电池：
+
image::../media/drw_g_nv_battery_replace_ieops-1864.svg[更换 NV 电池]

+
[cols="1,4"]
|===


 a| 
image::../media/icon_round_1.png[标注编号 1]
 a| 
将 NV 电池向上提起并从其隔间中取出。



 a| 
image::../media/icon_round_2.png[标注编号 2]
 a| 
从固定器上拆下线束。



 a| 
image::../media/icon_round_3.png[标注编号 3]
 a| 
.. 推入并按住连接器上的卡舌。
.. 将连接器向上拉出插座。
+
拉起时，从头到尾（纵向）轻轻摇动连接器以将其卸下。



|===
. 拆开更换用 NV 电池的包装，并将其放置在存储系统附近的无静电平坦表面上。
+
保存包装材料，以便在退回故障 NV 电池时使用。

. 安装更换后的 NV 电池：
+
.. 将接线连接器插入其插座。
.. 将电线沿电源侧布线，插入其固定器，然后穿过 NV 电池舱前面的通道。
.. 将 NV 电池放入其隔间。
+
NV 电池应平放在其隔间内。
