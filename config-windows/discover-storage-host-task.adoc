= 发现主机上的存储
:icons: font
:imagesdir: ../media/


[role="lead"]
添加新 LUN 时，必须手动重新扫描关联磁盘以发现它们。主机不会自动发现新的 LUN 。

存储系统上的 LUN 在 Windows 主机中显示为磁盘。

.步骤
. 以管理员身份登录。
. 要发现存储，请从 Windows 命令提示符处运行以下命令。
+
[listing]
----
# echo rescan | diskpart
----
. 要验证是否添加了新存储，请运行以下命令。
+
[listing]
----
# echo list disk | diskpart
----
