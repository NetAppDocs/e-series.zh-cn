= 配置交换机
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以根据供应商针对 NVMe over RoCE 的建议配置交换机。这些建议可能包括配置指令以及代码更新。


NOTE: * 数据丢失风险。 * 您必须在交换机上启用优先级流量控制或全局暂停控制，以消除基于 RoCE 的 NVMe 环境中的数据丢失风险。

.步骤
. 作为最佳实践配置，启用以太网暂停帧流量控制 * 端到端 * 。
. 有关为您的环境选择最佳配置的提示，请咨询您的网络管理员。
