= 验证主机上的存储访问
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在使用命名空间之前，请确认主机可以将数据写入命名空间并将其读回。

.您需要的内容
已初始化的命名空间，使用文件系统进行格式化。

.步骤
. 在主机上，将一个或多个文件复制到磁盘的挂载点。
. 将文件复制回原始磁盘上的其他文件夹。
. 运行 `dIFF` 命令将复制的文件与原始文件进行比较。


.完成后
删除复制的文件和文件夹。
