= 安装硬件 - EF50 和 EF80
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
准备安装 EF50 或 EF80 存储系统后，将机架安装硬件安装在机柜或电信机架中，然后安装并固定您的存储系统。

如果您的存储系统位于机柜中，则可以跳过此过程并转到 link:install-cable.html["为硬件布线"]。

.开始之前
* 准备一个无静电的平面工作区。
* 获取 ESD 腕带并采取防静电预防措施。
* 确保已将安装说明与机架安装硬件包装在一起。如果没有，请参见link:../rackmount-hardware.html["机架式硬件"^]。
* 请注意与存储系统重量相关的吊装预防措施。请参阅link:install-prepare.html#step-2-understand-the-lifting-precautions["了解起重预防措施"^]。
* 了解通过存储系统的气流从安装驱动器的前面进入，并从端口所在的后面排出。


.步骤
. 使用机架安装硬件附带的说明为您的存储系统安装机架安装硬件。
+

CAUTION: 将机架安装硬件安装在机柜或机架的最低可用空间中，以防止存储系统在安装时翻倒。

. 在机柜或电信机架中安装并固定您的存储系统：
+
.. 将存储系统放置在导轨上，然后从底部支撑存储系统，将其滑入到位。
.. 使用随附的安装螺钉将存储系统固定到机柜或电信机架上。


. 将挡板连接到存储系统前面。
+
.. 将挡板放置在存储系统的前面，以便两端的孔与存储系统上的紧固件对齐。
.. 将挡板卡入到位。




.下一步是什么？
为您的存储系统安装硬件后，link:install-cable.html["连接硬件电缆"]。
