= 安装和设置 - EF50 和 EF80
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要安装和设置 EF50 或 EF80 存储系统，必须准备好站点，安装并连接硬件组件，打开存储系统的电源，并完成存储系统设置和配置。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一个"]link:install-prepare.html["准备安装"]
[role="quick-margin-para"]
要准备安装，请准备好现场，了解存储系统的吊装注意事项，打开包装箱并将内容与装箱单进行比较，然后注册存储系统以获得支持优势。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["两个"]link:install-hardware.html["安装硬件"]
[role="quick-margin-para"]
要安装硬件，请在机柜或机架中安装机架安装硬件，然后安装并固定您的存储系统。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三个"]link:install-cable.html["为硬件布线"]
[role="quick-margin-para"]
要连接硬件，请将控制器彼此连接以启用控制器间镜像，这可确保完整的系统冗余，并用于缓存镜像和 I/O 传输，然后将控制器连接到主机网络。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["四个"]link:install-power-hardware.html["打开存储系统电源"]
[role="quick-margin-para"]
要为您的存储系统供电，请将电源线插入每个控制器电源，然后将它们连接到不同电路上的电源

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["五个"]link:install-complete-setup.html["完成存储系统设置"]
[role="quick-margin-para"]
要完成存储系统设置，请连接和配置控制器上的管理端口，配置和管理存储系统，然后验证主机 I/O 模块协议是否正确。
